# Babysitter (VS Code Extension)

Babysitter is a VS Code extension for orchestrating and monitoring `o` runs
from inside your editor. It adds a Runs tree view, command palette actions,
and webviews for run details and logs.

For full documentation, demos, and SDK/CLI walkthroughs, see:

- https://github.com/a5c-ai/babysitter
- https://github.com/a5c-ai/babysitter/blob/main/USER_GUIDE.md
