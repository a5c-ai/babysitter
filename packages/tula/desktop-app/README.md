# @a5c-ai/tula-desktop-app

Electron desktop application for Tula agent control and monitoring.

## Stack
- Electron 36 + Vite + React 19
- Shared components from `@a5c-ai/tula-ui`
- electron-builder for packaging

## Development
```bash
npm run dev --workspace=@a5c-ai/tula-desktop-app
```

## Build
```bash
npm run build --workspace=@a5c-ai/tula-desktop-app
```
