# babysitter:observe

## Purpose

Open observer workflow for run visibility and monitoring.

## Usage

```text
babysitter observe [watch target]
```

## Example

```text
babysitter observe current workspace
```

## Notes

- Use command phrases in Codex chat (`babysitter ...`), not custom slash commands.
