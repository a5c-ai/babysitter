# Iconography System Agent

Manage and validate icon systems for design consistency.

## Overview

This agent ensures icon libraries maintain visual consistency, accessibility compliance, and optimal performance across products.

## Key Capabilities

- **Consistency Validation**: Style guide enforcement
- **Grid Standards**: Size and alignment verification
- **SVG Optimization**: Performance optimization
- **Accessibility**: Label and contrast checking

## When to Use

- Auditing existing icon libraries
- Creating icon documentation
- Enforcing icon style guides
- Optimizing icon performance
