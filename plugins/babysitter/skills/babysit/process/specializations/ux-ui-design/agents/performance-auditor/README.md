# Performance Auditor Agent

Analyze UI/UX performance metrics and Core Web Vitals.

## Overview

This agent specializes in measuring and optimizing UI performance, with focus on Core Web Vitals and user-perceived performance metrics.

## Key Capabilities

- **Core Web Vitals**: LCP, FID/INP, CLS measurement
- **Budget Validation**: Check against performance budgets
- **Asset Analysis**: Identify optimization opportunities
- **Animation Profiling**: Frame rate and smoothness analysis

## When to Use

- Pre-launch performance audits
- Continuous performance monitoring
- Identifying performance regressions
- Optimizing user experience
