# Journey Map Visualizer Agent

Create and visualize user journey maps with emotional curves.

## Overview

This agent creates comprehensive journey maps that visualize the complete customer experience, highlighting emotional states, pain points, and opportunities.

## Key Capabilities

- **Touchpoint Mapping**: Multi-channel experience tracking
- **Emotional Curves**: Visualize sentiment across journey
- **Service Blueprints**: Backstage process mapping
- **Opportunity Identification**: Highlight improvement areas

## When to Use

- Mapping current customer experiences
- Designing future state experiences
- Creating service blueprints
- Identifying experience gaps
