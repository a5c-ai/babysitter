# Wireframe Generator Agent

Generate and iterate on wireframe designs at various fidelity levels.

## Overview

This agent creates wireframe designs from low to medium fidelity, with responsive variants and detailed annotations for design handoff.

## Key Capabilities

- **Multi-Fidelity**: Low, medium, and high fidelity wireframes
- **Responsive Variants**: Mobile, tablet, desktop versions
- **Annotations**: Detailed specifications and notes
- **Pattern Library**: Apply common UI patterns

## When to Use

- Early design exploration
- Rapid prototyping concepts
- Stakeholder alignment
- Developer handoff documentation
