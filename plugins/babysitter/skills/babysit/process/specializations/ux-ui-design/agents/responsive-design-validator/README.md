# Responsive Design Validator Agent

Validate responsive design across devices and breakpoints.

## Overview

This agent validates responsive implementations, ensuring designs work correctly across all target devices, breakpoints, and orientations.

## Key Capabilities

- **Cross-Device Testing**: Mobile, tablet, desktop validation
- **Breakpoint Analysis**: Transition behavior verification
- **Touch Target Validation**: Mobile usability checks
- **Overflow Detection**: Horizontal scroll issues

## When to Use

- QA testing responsive implementations
- Validating breakpoint behavior
- Mobile usability verification
- Design review automation
