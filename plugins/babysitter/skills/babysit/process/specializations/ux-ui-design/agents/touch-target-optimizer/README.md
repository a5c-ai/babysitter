# Touch Target Optimizer Agent

Optimize touch interactions for mobile and tablet devices.

## Overview

This agent validates and optimizes touch targets and mobile interactions, ensuring usability and accessibility on touch devices.

## Key Capabilities

- **Size Validation**: 44x44px minimum checks
- **Spacing Analysis**: Prevent accidental taps
- **Thumb Zone Mapping**: Reachability analysis
- **Gesture Patterns**: Interaction recommendations

## When to Use

- Mobile UI accessibility audits
- Touch interaction optimization
- Mobile-first design validation
- Gesture conflict detection
