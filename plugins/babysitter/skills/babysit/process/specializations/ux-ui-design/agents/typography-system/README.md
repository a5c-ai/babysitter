# Typography System Agent

Design and validate typography systems for design systems.

## Overview

This agent creates comprehensive typography systems with proper scales, pairings, and responsive behavior for consistent text styling across products.

## Key Capabilities

- **Type Scale Generation**: Modular scale calculations
- **Font Pairing**: Complementary font recommendations
- **Readability Optimization**: Line height and spacing
- **Responsive Typography**: Fluid sizing rules

## When to Use

- Establishing design system typography
- Creating typography documentation
- Optimizing readability
- Implementing responsive text
