# Zep Memory Integration

Integrates Zep memory server for long-term conversation memory and user profiling.

## Overview

This skill provides Zep memory integration, enabling persistent conversation memory with automatic summarization and entity extraction.

## Key Features

- Session management
- Automatic summarization
- Entity extraction
- Semantic memory search
- LangChain integration

## Usage

Use this skill when building conversational systems that require long-term memory and user context retention.
