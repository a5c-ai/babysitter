# Semantic Kernel Setup

Configures Microsoft Semantic Kernel for orchestrated AI with planners and plugins.

## Overview

This skill provides Semantic Kernel setup capabilities, enabling AI orchestration through plugins, planners, and memory integration.

## Key Features

- Plugin system configuration
- Multiple planner types
- Memory connectors
- Function calling
- Multi-provider support

## Usage

Use this skill when building AI applications using Microsoft's Semantic Kernel framework.
