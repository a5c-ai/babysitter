# Redis Memory Backend

Configures Redis backend for conversation state persistence and caching.

## Overview

This skill provides Redis integration for conversation memory, enabling persistent message storage, LLM caching, and real-time state management.

## Key Features

- Message history persistence
- LLM response caching
- TTL-based expiration
- Real-time Pub/Sub
- Connection pooling

## Usage

Use this skill when building conversational systems that require Redis-based memory persistence and caching.
