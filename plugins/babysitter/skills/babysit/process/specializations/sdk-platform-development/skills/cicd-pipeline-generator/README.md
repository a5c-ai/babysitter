# CI/CD Pipeline Generator

Generate CI/CD pipelines for SDK build and release.

## Purpose

This skill generates CI/CD pipelines for building, testing, and releasing SDKs across multiple languages and platforms.

## Key Features

- Multi-platform support
- Matrix builds
- Release automation
- Quality gates
- Security scanning

## When to Use

- Setting up SDK CI/CD
- Configuring release workflows
- Implementing quality gates
- Automating publishing
