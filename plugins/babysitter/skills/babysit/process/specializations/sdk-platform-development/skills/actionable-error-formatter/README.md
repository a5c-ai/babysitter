# Actionable Error Formatter

Format errors with actionable fix suggestions and documentation links.

## Purpose

This skill creates error messages that help developers quickly understand and resolve issues through actionable suggestions and relevant documentation.

## Key Features

- Contextual fix suggestions
- Documentation links
- Debug mode support
- Structured error output
- Request correlation

## When to Use

- Improving error messages
- Adding debug capabilities
- Linking errors to docs
- Formatting stack traces
