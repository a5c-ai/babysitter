# API Gateway Configurator

Configure API gateways for SDK traffic management.

## Purpose

This skill configures API gateways to handle routing, rate limiting, authentication, and resilience for reliable API delivery.

## Key Features

- Request routing
- Rate limiting
- Authentication setup
- Circuit breakers
- Request transformation

## When to Use

- Setting up API gateways
- Configuring rate limits
- Implementing API authentication
- Managing traffic routing
