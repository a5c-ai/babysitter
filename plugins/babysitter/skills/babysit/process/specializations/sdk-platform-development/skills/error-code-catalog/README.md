# Error Code Catalog

Manage and document SDK error codes and messages.

## Purpose

This skill maintains a comprehensive error code catalog ensuring consistent, actionable error handling across SDK operations.

## Key Features

- Error taxonomy management
- Auto-generated documentation
- Localization support
- Troubleshooting guides
- Quality validation

## When to Use

- Defining SDK error structure
- Documenting error codes
- Localizing error messages
- Improving error quality
