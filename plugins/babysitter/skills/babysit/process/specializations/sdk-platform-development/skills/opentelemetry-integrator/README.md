# OpenTelemetry Integrator

Integrate OpenTelemetry tracing and metrics into SDKs.

## Purpose

This skill adds comprehensive observability to SDKs through OpenTelemetry, enabling distributed tracing and metrics collection.

## Key Features

- Distributed tracing
- Metrics collection
- Context propagation
- Multi-exporter support
- Semantic conventions

## When to Use

- Adding observability to SDKs
- Implementing distributed tracing
- Collecting SDK metrics
- Debugging distributed systems
