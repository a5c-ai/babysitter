# Health Check Endpoint

Implement health check and readiness endpoints for SDK consumers.

## Purpose

This skill implements health check endpoints for verifying service connectivity and dependency health, essential for Kubernetes and load balancer integration.

## Key Features

- Kubernetes probe support
- Dependency health checks
- Graceful degradation
- Health aggregation
- Custom health indicators

## When to Use

- Configuring Kubernetes probes
- Monitoring service health
- Implementing load balancer checks
- Managing dependency failures
