# Middleware Chain Designer

Design middleware and interceptor chains for SDK extensibility.

## Purpose

This skill designs middleware architectures that enable SDK extensibility through pluggable request/response processing chains.

## Key Features

- Middleware interfaces
- Interceptor chains
- Before/after hooks
- Custom transports
- Built-in middleware

## When to Use

- Designing SDK extensibility
- Implementing request interceptors
- Building custom transports
- Adding middleware composition
