# Critical Thinking Educator Agent

A specialized agent for designing and delivering critical thinking instruction and assessment.

## Overview

The Critical Thinking Educator Agent provides expert guidance on teaching critical thinking skills. It assists with curriculum design, instructional delivery, assessment development, cognitive bias awareness, and fostering reasoning capabilities in educational and professional contexts.
