# Clinical Informatics Specialist Agent

A specialized agent for implementing and optimizing health information technology systems to support clinical operations.

## Overview

The Clinical Informatics Specialist Agent provides expert guidance on health IT implementation and optimization. It assists with EHR configuration, clinical decision support, data integration, and workflow optimization to improve clinical effectiveness.
