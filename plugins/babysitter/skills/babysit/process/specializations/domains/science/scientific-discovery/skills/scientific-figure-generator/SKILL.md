# Scientific Figure Generator Skill

## Purpose

Generate publication-quality scientific figures following journal standards and best practices for data visualization.

## Capabilities

- Create statistical plots
- Generate multi-panel figures
- Apply journal formatting
- Export in multiple formats
- Ensure accessibility
- Add annotations

## Usage Guidelines

1. Prepare data
2. Select visualization type
3. Apply styling
4. Add annotations
5. Check accessibility
6. Export figure

## Process Integration

Works within scientific discovery workflows for:
- Manuscript preparation
- Presentation creation
- Data communication
- Results visualization

## Configuration

- Journal style guides
- Color schemes
- Resolution settings
- Export formats

## Output Artifacts

- Publication figures
- Multi-panel layouts
- Vector graphics
- Figure legends
