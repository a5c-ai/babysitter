# Mobile Offline Storage Skill

## Overview

The Mobile Offline Storage skill provides cross-platform offline-first data management capabilities. It enables configuration of various storage solutions and sync architectures for mobile apps.

## Key Capabilities

- **WatermelonDB**: React Native offline database
- **Realm**: Cross-platform persistence with sync
- **MMKV**: Fast key-value storage
- **Sync Queue**: Offline-first sync patterns
- **Conflict Resolution**: Merge strategies

## Quick Start

### WatermelonDB Setup

```bash
npm install @nozbe/watermelondb
npm install @babel/plugin-proposal-decorators
```
