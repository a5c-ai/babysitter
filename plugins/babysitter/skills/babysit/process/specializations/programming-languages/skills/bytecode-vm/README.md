# Bytecode VM Skill

Expert skill for bytecode virtual machine design including instruction set design, dispatch mechanisms, and stack/register architectures.

## Quick Reference

- **Category**: Runtime
- **ID**: SK-010

## Key Capabilities

- Bytecode instruction set design
- Stack-based vs register-based VM implementation
- Efficient dispatch (switch, computed goto, threaded)
- Compact bytecode encoding design
- Bytecode verification
- Exception handling in bytecode
- Inline caching for dynamic dispatch
- Bytecode serialization/deserialization

## When to Use

Use this skill when implementing:
- Bytecode interpreters
- Virtual machine instruction sets
- Bytecode compilers
- Runtime performance optimization
