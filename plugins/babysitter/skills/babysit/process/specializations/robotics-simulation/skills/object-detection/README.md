# Object Detection/Segmentation Skill

Deep learning based object detection and segmentation for robotics applications.

## Purpose

Provides expert capabilities for deploying and optimizing deep learning perception models on robot platforms, including real-time optimization for edge devices.

## Key Features

- YOLO configuration and deployment
- Instance segmentation with Detectron2
- TensorRT/ONNX optimization
- 3D object detection
- Multi-object tracking
- Multi-camera fusion

## When to Use

- Deploying detection models on robots
- Optimizing inference for Jetson/edge
- Implementing object tracking
- Setting up 3D object detection
- Fusing detections from multiple cameras
